# BevelFilter

PixiJS filter to apply a bevel effect.

## Installation

```bash
npm install @pixi/filter-bevel
```

## Usage

```js
import {BevelFilter} from '@pixi/filter-bevel';
import {Container} from 'pixi.js';

const container = new Container();
container.filters = [new BevelFilter()];
```
