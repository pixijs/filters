# GlowFilter

PixiJS filter to apply a glow effect.

## Installation

```bash
npm install @pixi/filter-glow
```

## Usage

```js
import {GlowFilter} from '@pixi/filter-glow';
import {Container} from 'pixi.js';

const container = new Container();
container.filters = [new GlowFilter()];
```
