# GodrayFilter

PixiJS v4 filter for godray effect

## Installation

```bash
npm install @pixi/filter-godray
```

## Usage

```js
import {GodrayFilter} from '@pixi/filter-godray';

const container = new PIXI.Container();
container.filters = [new GodrayFilter()];
```
