# ZoomFilter

PixiJS v4 filter to render Zoom Blur effect.

## Installation

```bash
npm install @pixi/filter-zoom-blur
```

## Usage

```js
import { ZoomFilter } from '@pixi/filter-zoom-blur';

const container = new PIXI.Container();
container.filters = [new ZoomFilter()];
```
